= Saving generated policies

[role="_abstract"]
You can download and save the generated network policies from {product-title}.
Use this option to commit the policies into a version control system like Git.

.Procedure
* Select the *Download YAML* icon on the *Network Policy Simulator* panel.
+
[NOTE]
====
. To create policies using the saved YAML file, use the following command:

* On {ocp}:
+
[source,terminal]
----
$ oc create -f "StackRox Generated.yaml"
----
* On Kubernetes:
+
[source,terminal]
----
$ kubectl create -f "StackRox Generated.yaml"
----
. If the generated policies cause problems, you can remove them by running the following command:
* On {ocp}:
+
[source,terminal]
----
$ oc delete -f "StackRox Generated.yaml"
----
* On Kubernetes:
+
[source,terminal]
----
$ kubectl delete -f "StackRox Generated.yaml"
----
====
