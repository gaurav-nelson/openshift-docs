= Finding Kubernetes roles are their assignment

[role="_abstract"]
Use the *Configuration Management* view to get information about which Kubernetes roles are assigned to which users and groups.

.Procedure
. Navigate to the RHACS portal and click *Configuration Management* from the left-hand navigation menu.
. Select *RBAC Visibility and Configuration* -> *Users and Groups* from the *Configuration Management* view header.
The *Users and Groups* view displays a list of Kubernetes users and groups, their assigned roles, and whether the `cluster-admin` role is enabled for each of them.
. Select a row to view more details about cluster and namespace permissions associated with the selected user or group.
The information appears in a panel on the right.
